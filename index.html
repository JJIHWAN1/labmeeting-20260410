데이터 스무딩

가중 이동평균 (Weighted Moving Average)

데이터 스무딩 — 이동평균

10분 간격 계단식 데이터를 부드러운 곡선으로 변환

📊 왜 필요한가?

  • 데이터가 10분마다 갱신 → 계단식 그래프
  • 이동평균은 일정 기간 동안의 평균을 선으로 연결
  • 주식/코인의 기술적 분석 지표와 동일한 원리
이동평균 = 노이즈를 제거하고
전체 추세를 파악하기 위한 기법

⚙️ 적용 방식

  • smoothData() 함수 — 가중 이동평균
  • 윈도우 크기: windowSize = 7 (앞뒤 3개씩 ≈ 70분)
  • 중앙값에 가장 높은 가중치, 양쪽 끝으로 갈수록 감소
원래 데이터 (계단)
14000 → 14000 → 8000 → 8000
스무딩 후 (곡선)
14000 → 12300 → 9600 → 8400

🎯 적용 대상

인구 최소/최대 도로 속도 기온 습도 PM2.5 PM10 주차 따릉이
+ Chart.js tension: 0.4와 결합하여 스무스한 곡선으로 표시

배포 환경 변경

Netlify → Cloudflare Pages 마이그레이션

배포 환경 변경: Netlify → Cloudflare Pages

무료 크레딧 제한 → 대역폭 무제한 플랫폼으로 이전

❌ Netlify 문제점

  • 무료 플랜 대역폭: 100GB/월 제한
  • 크레딧 소진 시 사이트 전체 정지 (503 에러)
  • Scheduled Function도 함께 중단 → 데이터 수집 불가
  • 매월 초 리셋 대기 or 유료 전환($19/월) 필요

✅ Cloudflare Pages 장점

  • 대역폭: 무제한 (Unlimited)
  • Workers 무료: 10만 회/일
  • 글로벌 CDN으로 응답 속도 빠름
  • 기존 Supabase DB 그대로 유지 (URL·키 동일)
